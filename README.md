<h1>The Frontend of Hotwheels</h1>
<p>Hotwheels is an e-commerce platform built with React js,Node js and GraphQL written with react and typescript </p>

## Team Members :

Rajat Vohra (rajatvohra), Abhyudaya Prakash Rai (BetterThanBlank), Rishabh Varshney (Rishabh Varshney), Vrinda Kakkar (Vrinda-Kakkar)

## TECHNOLOGIES/TOOLS USED

- TYPESCRIPT
- REACT
- GRAPHQL
- APOLLO
- NEST.JS
- POSTGRESQL
- HTML/CSS
- TYPEORM
